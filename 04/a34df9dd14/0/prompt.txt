## 1. **HumanEval** - OpenAI's Foundational Benchmark

**Overview:** The gold-standard benchmark for functional code generation, released by OpenAI in 2021 alongside the Codex model.

**Dataset Specs:**
- **164 hand-crafted programming problems** (function-level)
- **Python-focused** (original version)
- **Difficulty:** Entry to intermediate level programming
- **Test cases per problem:** ~7.7 on average

**Metric:**
- **Pass@k:** The probability that at least one of the top k-generated samples passes all unit tests
  - Pass@1: First solution passes
  - Pass@10: At least one of 10 attempts passes
  - Pass@100: At least one of 100 attempts passes
- Evaluates **functional correctness** (more practical than syntactic matching)

**How to Run:**
- **Official repository:** [openai/human-eval](https://github.com/openai/human-eval)
- Installation: `pip install -e human-eval`
- Requires Python 3.7+
- **Critical security note:** Execution is commented out by default since it runs untrusted model code. Must be explicitly enabled with proper sandboxing.

**Code Execution:** YES - requires actual code execution and test harness

**Typical Scores (pass@1):**
- Claude 3.5 Sonnet: 92%
- GPT-4o: 91%
- Claude 3 Opus: 84.9%
- Llama 3.1 405B: 89%
- Code Llama 70B: 67.8%

**Access:** [github.com/openai/human-eval](https://github.com/openai/human-eval), [Hugging Face versions available](https://huggingface.co/datasets/openai_humaneval)

**Overview:** Addresses weaknesses in the original HumanEval by adding significantly more test cases to catch incorrect solutions.

**Performance Impact:**

**Metric:** Same pass@k as HumanEval, but much more reliable
