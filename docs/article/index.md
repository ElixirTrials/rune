# Scientific Article

This section contains the scientific article documenting the theoretical foundation
and algorithmic design of Rune's parametric episodic memory approach.

<div class="abstract" markdown="1">
**Abstract.** This article presents the design of Rune, a local coding agent that
builds persistent parametric memory through composable LoRA adapters trained on its
own coding trajectories. *Content forthcoming in subsequent phases.*
</div>
