**Implement calculate_pass_at_k()** in metrics.py — replace the NotImplementedError stub:
- Use the unbiased estimator from Chen et al. 2021: `1 - prod((n-c-i)/(n-i) for i in range(k))`
- Import `math` (stdlib only — no GPU deps)
- Guard: if n_correct > n_samples: raise ValueError
- Guard: if n_correct == n_samples: return 1.0
- Guard: if n_samples - n_correct < k: return 1.0
- Use `math.prod()` for the product form (numerically stable, avoids large factorial)

**Implement run_kill_switch_gate()** as a NEW function in metrics.py:
- Signature: `run_kill_switch_gate(baseline_pass1: float, adapter_pass1: float, threshold: float = 0.05) -> dict[str, object]`
- Compute relative_delta: `(adapter_pass1 - baseline_pass1) / max(baseline_pass1, 1e-9)` (1e-9 prevents div-by-zero)
- verdict = "PASS" if `adapter_pass1 >= baseline_pass1 * (1 + threshold)` else "FAIL"
- Print human-readable summary with PASS/FAIL, baseline, adapter, delta
- Return dict with baseline_pass1, adapter_pass1, relative_delta, verdict
