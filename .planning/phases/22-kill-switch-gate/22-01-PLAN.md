**DocToLoraHypernetwork class** (nn.Module):
- Constructor params: `input_dim` (token vocab size for embedding, NOT the 7B model embedding — hypernetwork learns its own), `num_latents=32`, `latent_dim=256`, `depth=4`, `heads=8`, `rank=8`, `target_modules=("q_proj", "v_proj")`, `num_layers=28` (Qwen2.5-7B layer count), `hidden_dim=4096` (q_proj/v_proj hidden dim)
- `self.token_embedding = nn.Embedding(input_dim, latent_dim)` — the hypernetwork learns its own lightweight embedding from token IDs (avoids loading the 7B base model)
- `self.latents = nn.Parameter(torch.randn(num_latents, latent_dim))` — learnable latent array
- `self.cross_attend = nn.MultiheadAttention(latent_dim, heads, batch_first=True)` — latents as Q, token embeddings as K/V
- `self.self_attend = nn.TransformerEncoder(nn.TransformerEncoderLayer(latent_dim, heads, batch_first=True), num_layers=depth)` — process latents
- `self.weight_head = nn.Linear(latent_dim * num_latents, len(target_modules) * num_layers * 2 * rank * hidden_dim)` — project to LoRA weights (2 = A+B)
- Store `rank`, `hidden_dim`, `target_modules`, `num_layers` as instance attrs for reshaping in forward()

**forward(self, token_ids: torch.Tensor) -> dict[str, torch.Tensor]:**
- token_ids shape: (batch, seq_len) — integer token IDs
- Embed: `x = self.token_embedding(token_ids)` -> (batch, seq_len, latent_dim)
- Expand latents for batch: `latents = self.latents.unsqueeze(0).expand(batch_size, -1, -1)`
- Cross-attention: `latents, _ = self.cross_attend(latents, x, x)`
- Self-attention: `latents = self.self_attend(latents)`
- Flatten: `latents = latents.reshape(batch_size, -1)` -> (batch, num_latents * latent_dim)
- Project: `weights = self.weight_head(latents)` -> (batch, total_weight_params)
- Reshape into PEFT state_dict: iterate over layers and target_modules, slice weights into lora_A (rank, hidden_dim) and lora_B (hidden_dim, rank) for each
- Return dict with keys following PEFT naming: `base_model.model.model.layers.{i}.self_attn.{module}.lora_A.weight` and `.lora_B.weight`
- Note: return the FIRST batch element (squeeze batch dim) since hypernetwork generates one adapter per trajectory

**save_hypernetwork_adapter(weights, output_dir, base_model_id, rank=8, target_modules=None):**
- Deferred imports: `from safetensors.torch import save_file`
- Default target_modules to ["q_proj", "v_proj"]
- Save weights dict to `adapter_model.safetensors` via save_file()
- Write `adapter_config.json` with fields: peft_type="LORA", r=rank, lora_alpha=rank*2, target_modules, lora_dropout=0.0, bias="none", task_type="CAUSAL_LM", base_model_name_or_path=base_model_id, inference_mode=True, modules_to_save=None, fan_in_fan_out=False
- CRITICAL: Do NOT include embed_tokens or lm_head weights — vLLM rejects these (per Phase 21-01 decision)
