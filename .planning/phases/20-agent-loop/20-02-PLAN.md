**Helper functions (private):**

`_build_prompt(state: RuneState) -> str`:
- First attempt (attempt_count == 0): task_description + test_suite + instruction to write solution
- Retry attempt (attempt_count > 0): same + prior generated_code + stdout + stderr + exit_code with "Your previous attempt produced the following errors:" prefix

`_extract_code(text: str) -> str`:
- `re.search(r"```python\s*(.*?)```", text, re.DOTALL)`
- If match: return `match.group(1).strip()`
- Fallback: return `text.strip()`

**generate_node(state) -> dict[str, Any]:**
- Get provider via `get_provider()` (not at module level -- call inside function)
- Build prompt via `_build_prompt(state)`
- Read model from env var `RUNE_MODEL` with fallback `DEFAULT_MODEL` (inside function body for monkeypatch)
- Determine adapter_id: `state["adapter_ids"][0] if state["adapter_ids"] else None`
- Call `await provider.generate(prompt=full_prompt, model=model, adapter_id=adapter_id)`
  - full_prompt should be system prompt + user prompt concatenated, OR pass system_prompt separately -- since InferenceProvider.generate() takes a single `prompt` param, concatenate: `f"{SYSTEM_PROMPT}\n\n{user_prompt}"`
- Extract code via `_extract_code(result.text)`
- Return `{"generated_code": extracted_code}`
- Log the generation at INFO level

**execute_node(state) -> dict[str, Any]:**
- Read timeout from env var `RUNE_EXEC_TIMEOUT` with fallback `DEFAULT_TIMEOUT` (inside function body)
- Concatenate: `state["generated_code"] + "\n\n" + state["test_suite"]`
- Use `tempfile.TemporaryDirectory()` context manager (auto-cleanup per Claude's discretion)
- Write concatenated script to `{tmpdir}/solution.py`
- Run `subprocess.run(["python", script_path], capture_output=True, text=True, timeout=timeout, cwd=tmpdir)`
- Return `{"stdout": proc.stdout, "stderr": proc.stderr, "exit_code": proc.returncode, "tests_passed": proc.returncode == 0}`
- Handle `subprocess.TimeoutExpired`: return `{"stdout": "", "stderr": f"Execution timed out after {timeout}s", "exit_code": 1, "tests_passed": False}`
- Log execution result at INFO level

**reflect_node(state) -> dict[str, Any]:**
- Build step dict: `{"generated_code": ..., "stdout": ..., "stderr": ..., "exit_code": ..., "tests_passed": ...}` from state
- Return `{"attempt_count": state["attempt_count"] + 1, "trajectory": state["trajectory"] + [step]}`
  - CRITICAL: Use `state["trajectory"] + [step]` (new list), NOT `.append()` -- LangGraph requires immutable state updates
- Log at INFO level with attempt number

**save_trajectory_node(state) -> dict[str, Any]:**
- Determine outcome: `"success" if state["tests_passed"] else "exhausted"`
- Call `record_trajectory(session_id=state["session_id"], steps=state["trajectory"], outcome=outcome, task_description=state["task_description"], task_type=state["task_type"], adapter_ids=state["adapter_ids"])`
- Return `{"outcome": outcome}`
- Log at INFO level with session_id and outcome
