    1. `D2LTrainConfig(BaseModel)` Pydantic model per CONTEXT.md spec:
       - Fields: base_model_name (str, default "Qwen/Qwen3-Coder-Next"), sakana_checkpoint_path (str), num_steps (int, 100), lr (float, 2e-4), alpha (float, 0.5), temperature (float, 2.0), checkpoint_every (int, 100), full_checkpoint_every (int, 500), checkpoint_dir (str, "./checkpoints"), experiment_name (str, "d2l-qwen3"), dry_run (bool, False), smoke_test (bool, False), dataset_path (str | None, None), grad_clip (float, 1.0), warmup_steps (int, 10), lora_r (int, 8), max_length (int, 512)

       - Signature: `(student_logits: torch.Tensor, teacher_logits: torch.Tensor, answer_start: int, config: D2LTrainConfig) -> tuple[torch.Tensor, dict[str, float]]`
       - Slice both logits to `[:, answer_start:, :]` for answer span masking
       - KL: `F.kl_div(F.log_softmax(s/T, dim=-1), F.softmax(t/T, dim=-1), reduction="batchmean")` — MUST use reduction="batchmean"
       - CE: `F.cross_entropy(s.reshape(-1, vocab), t.argmax(-1).reshape(-1))`
       - Blend: `total = alpha * kl + (1 - alpha) * ce`
       - Return `(total_loss, {"kl_loss": kl.item(), "ce_loss": ce.item(), "total_loss": total.item()})`

    3. Module boilerplate: `from __future__ import annotations`, `__all__ = ["train_d2l_qwen3", "D2LTrainConfig"]`, logger, Google-style docstrings
    4. Stub `def train_d2l_qwen3(config: D2LTrainConfig) -> dict[str, Any]: ...` with `raise NotImplementedError("Implemented in Plan 02")` — needed for __init__.py export to resolve
