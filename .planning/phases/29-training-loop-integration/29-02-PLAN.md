  <name>Task 1: Training loop, checkpoint, MLflow, modes, CLI</name>
  <files>libs/model-training/src/model_training/d2l_train.py</files>
  <action>
    Extend d2l_train.py (which already has D2LTrainConfig and _compute_kl_ce_loss from Plan 01) with the following functions. ALL GPU-dependent imports (torch, transformers, mlflow, etc.) MUST be deferred inside function bodies with `# noqa: PLC0415` per project pattern (INFRA-05).

    1. **`_training_step(record, base_model, tokenizer, hypernet, hc, config) -> tuple[torch.Tensor, dict[str, float]]`**
       Two-pass teacher/student separation (TRAIN-02):
       - Pass 1: `features, attn_mask = extract_activations_with_model(text=record["activation_text"], model=base_model, tokenizer=tokenizer, layer_indices=list(hc.layer_indices), max_length=config.max_length)` — context only, NO answer tokens
       - Hypernetwork forward (OUTSIDE torch.no_grad): `lora_dict, _ = hypernet.generate_weights(features, attn_mask, None)`
       - Pass 2 teacher: tokenize record["teacher_text"], run base_model with `torch.no_grad()`, get teacher_logits
       - Pass 2 student: compute answer_start = len(tokenizer(record["activation_text"], truncation=True, max_length=config.max_length)["input_ids"]), run base_model inside `apply_functional_lora(base_model, lora_dict, hc)` context with same teacher_inputs, get student_logits
       - Return `_compute_kl_ce_loss(student_logits, teacher_logits, answer_start, config)`
       - Keep base_model.eval() throughout. Hypernet stays in train() mode.

    2. **`_save_checkpoint(step, hypernet, optimizer, scheduler, config, hc, best_loss, full=False) -> Path`** (TRAIN-05)
       - Tiered: lightweight (hypernet state_dict, config.model_dump() as config_json, step, attention_layer_indices from hc, best_loss) every checkpoint_every steps
       - Full: adds optimizer_state_dict, scheduler_state_dict, torch.get_rng_state(), cuda rng if available — every full_checkpoint_every steps
       - Format: `torch.save(payload, ckpt_dir / f"ckpt-{step}.pt")`
       - Return the checkpoint path

    3. **`_setup_mlflow(config: D2LTrainConfig) -> None`**
       - Set tracking URI from `MLFLOW_TRACKING_URI` env var or default "./mlruns"
       - Set experiment name from config.experiment_name

    4. **`train_d2l_qwen3(config: D2LTrainConfig) -> dict[str, Any]`** — REPLACE the NotImplementedError stub from Plan 01
       - Mode dispatch: if config.dry_run → call `_dry_run_validate_shapes(config)` and return; if config.smoke_test → set config.num_steps = min(config.num_steps, 5)
       - Setup: deferred imports of torch, transformers, mlflow inside function body
       - Load tokenizer: `AutoTokenizer.from_pretrained(config.base_model_name)`
       - Load base model: `AutoModelForCausalLM.from_pretrained(config.base_model_name, output_hidden_states=True)` → `.eval()`
       - Build hypernet config: `build_qwen3_hypernet_config(aggregator_config=get_aggregator_config(config.sakana_checkpoint_path))`
       - Create hypernet and transfer: `transfer_aggregator_weights(hypernet, config.sakana_checkpoint_path)` → `.train()`
       - Detect device: cuda if available, else cpu
       - Move models to device
       - Optimizer (TRAIN-04): `trainable_params = [p for p in hypernet.parameters() if p.requires_grad]`, `AdamW(trainable_params, lr=config.lr)`
       - Scheduler: `SequentialLR([LinearLR(start_factor=0.01, total_iters=warmup_steps), CosineAnnealingLR(T_max=max(1, num_steps - warmup_steps), eta_min=1e-6)], milestones=[warmup_steps])`
       - Data: for smoke_test/dry_run use `generate_needle_dataset()`, for full training use `load_jsonl(config.dataset_path)` → `split_by_task_id()`
       - MLflow: `_setup_mlflow(config)`, wrap loop in `mlflow.start_run(run_name=...)`, `mlflow.log_params(config.model_dump())`
       - Training loop: iterate records cycling if num_steps > len(records), call `_training_step`, `loss.backward()`, `clip_grad_norm_(trainable_params, config.grad_clip)`, `optimizer.step()`, `scheduler.step()`, `optimizer.zero_grad()`, log metrics via mlflow
       - Checkpoint: if step % checkpoint_every == 0 save lightweight, if step % full_checkpoint_every == 0 save full, log artifact to mlflow
       - Track best_loss, return summary dict with final_loss, best_loss, num_steps_completed, checkpoint_dir
       - For smoke_test: assert all losses finite and loss[-1] < loss[0]

    5. **`_dry_run_validate_shapes(config: D2LTrainConfig) -> dict[str, Any]`** (TRAIN-06)
       - Same setup as train (load model, hypernet, tokenizer)
       - Generate 1 needle record
       - Run single _training_step WITHOUT backward/optimizer
       - Assert features shape, lora_dict shapes, student_logits shape, teacher_logits shape
       - Log all shape validations, return shape summary dict, exit cleanly

    6. **CLI `if __name__ == "__main__":` block**
       - argparse with all D2LTrainConfig fields as CLI flags
       - `--dry-run` and `--smoke-test` as `store_true`
       - `--dataset` maps to dataset_path
       - `D2LTrainConfig(**vars(args))` → `train_d2l_qwen3(config)`

    ANTI-PATTERNS TO AVOID (from RESEARCH.md):
    - Do NOT call hypernet.generate_weights() inside torch.no_grad() — severs autograd
    - Do NOT use reduction='mean' or 'sum' for F.kl_div — MUST use 'batchmean'
    - Do NOT include answer tokens in activation_text pass
    - Do NOT put frozen aggregator params in optimizer
    - Do NOT call scheduler.step() before optimizer.step()
    - Do NOT set base_model to train() mode
